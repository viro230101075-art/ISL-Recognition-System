# 🤟 Indian Sign Language Recognition System

A real-time Indian Sign Language (ISL) Recognition System developed using Deep Learning and Computer Vision techniques to support assistive communication for people with hearing, speech, and certain motor impairments.

The system is designed to recognize both static hand gestures and dynamic gesture sequences in real time using hand landmark detection and sequence-based learning models.

---

# 🚀 Features

- Real-time hand gesture recognition
- Static alphabet/sign recognition
- Dynamic word and sequence recognition
- Lightweight and CPU-friendly implementation
- Human-centered assistive AI approach
- Personalized gesture adaptability for users with physical limitations
- Real-time inference using webcam input

---

# 🧠 Project Overview

This project focuses on building an intelligent assistive communication system capable of translating Indian Sign Language gestures into understandable outputs using AI-based recognition techniques.
